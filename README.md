# 🏠 House Price Prediction using Machine Learning

## 📌 Project Overview

This project predicts house prices using a Linear Regression Machine Learning model. The application is built using Python, Flask, HTML, and CSS.

## 🚀 Features

- Predict house prices
- Professional Flask Web UI
- Input Validation
- Linear Regression Model
- Responsive Design

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Flask
- HTML
- CSS
